An overview of my recent projects

  • AIFSB : AI for Sports Betting
    • Descriptive : An AI-based program that seeks for profitable betting strategies. This project, which I developed from scratch, is capable of identifying possible value bets in various sports and leagues. During the first 2 months of exploitation - brutally stopped by the Covid-19 pandemic -, betting on these selected bets yielded a Return On Investment (ROI) of 7.42% over 694 games, allowing me to earn a 4-digits amount, after starting from a 250 euros capital.

  • scrapeVIN : A python package for scraping vinted.fr
    • Descriptive : This pip package provides the user a set of tools to scrape vinted on a daily basis. The user can create a set of requests and criteria for which the code will look for on the website, and then sending the current available products via mail (mail part in development). The code has even been adopted for scraping foreign versions of the website : USA, UK, Germany, and multiple european countries.
