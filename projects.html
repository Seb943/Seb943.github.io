An overview of my recent projects

  • AIFSB : AI for Sports Betting
    • Descriptive : An AI-based program that seeks for profitable betting strategies. This project, which I developed from scratch, is capable of identifying possible value bets in various sports and leagues. During the first 2 months of exploitation - brutally stopped by the Covid-19 pandemic -, betting on these selected bets yielded a Return On Investment (ROI) of 7.42% over 694 games, allowing me to earn a 4-digits amount, after starting from a 250 euros capital.

  • TBpolo : A trading bot using Python and Poloniex API
    • Descriptive : Also an AI-based project, this program is a trading bot project construction operating on the Poloniex cryptocurrencies exchange. The principle used is, from the raw historical data of the Bitcoin on Poloniex (price, volumes, min, max), to compute financial predictive variables to estimate the probabilities of the asset either going up or down, and finally to create and identify profitable trading strategies. The process is interesting in the sense that the pipeline could be applied to any other financial market (stock market, raw materials, electricity, etc...). All steps are detailed in the report.

  • scrapeVIN : A python package for scraping vinted.fr
    • Descriptive : This python package provides the user a set of tools to scrape vinted on a daily basis.
